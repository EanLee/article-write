
- ❌ 資訊密度低

## 🧪 使用體驗對比

### 情境 1：瀏覽 50 篇文章

**舊版：**
```
1. 滾動 5500px 才能看完
2. 需要滾動 10-15 次
3. 眼睛疲勞
4. 系列文章分散難找
```

**新版：**
```
1. 只需滾動 1320px
2. 滾動 3-4 次即可
3. 系列文章集中顯示
4. 可摺疊不關心的系列
```

**結論：** 效率提升 **3-4 倍**

---

### 情境 2：編輯系列文章

**舊版：**
```
1. 看到「Vue 3 深入理解」
2. 想找同系列的其他文章
3. 需要記住系列名稱
4. 搜尋「Vue 3 進階」
5. 在結果中找到相關文章
```

**新版：**
```
1. 看到當前文章在「Vue 3 進階教學」分組
2. 直接在分組下看到其他文章
3. 一鍵切換到 #1 或 #2
```

**結論：** 操作步驟減少 **60%**

## 🎨 設計原則

### 1. 空間效率優先
- 單行顯示
- 去除不必要的邊距和陰影
- 可摺疊隱藏不需要的內容

### 2. 資訊層次清晰
- 系列 > 文章
- 圖示 > 文字
- 顏色編碼（當前/未儲存/已發布）

### 3. 熟悉的操作模式
- 類似 IDE 檔案瀏覽器
- 標準的展開/收合行為
- 鍵盤快捷鍵支援

### 4. 靈活可配置
- 視圖模式切換
- 圖示顯示開關
- 設定持久化
