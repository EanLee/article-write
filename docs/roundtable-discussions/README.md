## 🚀 如何開始討論

### 1. 提出議題

向 AI 助手提出問題或議題，例如：
```
我想討論一個問題：是否應該實作深色模式？
```

### 2. 多角色回應

AI 助手會以五個角色的視角進行分析和回應。

### 3. 延續討論

可以繼續提出問題或回應：
```
我同意技術長的觀點，但行銷的擔憂也有道理。
有沒有折衷方案？
```

### 4. 記錄與決策

討論會自動記錄到對應的話題目錄中，並形成決策文件。

## 💡 討論建議

### 適合討論的議題

✅ 新功能提議和評估
✅ 技術選型和架構決策
✅ 優先級排序和資源分配
✅ 使用者體驗改善
✅ 品牌和行銷策略
✅ 風險評估和問題解決

### 不適合的議題

❌ 純技術實作細節（應該在技術文件討論）
❌ 個人偏好（缺乏客觀標準）
❌ 已有明確答案的問題
❌ 與產品無關的議題
