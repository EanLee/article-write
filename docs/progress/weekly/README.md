# 週檢查點記錄

本目錄包含每週的自我檢查點記錄。

## 檢查點時間

**時間**：每週日晚上 20:00
**時長**：30 分鐘
**形式**：自我檢查 + 文檔記錄

## 檢查點目的

1. 回顧本週完成的工作
2. 評估下週工作量
3. 選擇下週工作模式（A/B/C）
4. 調整計畫（如需要）
5. 保持專案連續性

## 記錄格式

每週建立新檔案：`YYYY-MM-DD.md`

使用模板：`template.md`
