# 📁 IDE 風格文章樹使用指南

## 🌟 概述

全新的 IDE 風格文章列表提供了類似 VS Code 檔案瀏覽器的體驗，讓您能夠高效地瀏覽和管理文章，特別是系列文章。

---

## 🎨 視覺對比

### 舊版卡片式（已棄用）
```
每篇文章佔用 100-120px 高度
10 篇文章需要約 1000-1200px 垂直空間
```

### 新版樹狀式
```
每篇文章佔用 24px 高度
10 篇文章只需約 240px 垂直空間
節省 80% 空間！
```

---

## 🚀 主要功能

### 1. 系列分組視圖

文章會自動按系列分組：

```
📚 Vue 3 進階教學 (3)
  ● #1 · 基礎概念
  ✓ #2 · 進階應用
  ✓ #3 · 深入理解

📚 React 系列 (2)
  ✓ #1 · Hook 實戰
  * #2 · 性能優化

📄 獨立文章 (5)
  ○ TypeScript 技巧
  ○ 另一篇文章
```

**圖示說明：**
- `●` = 當前正在編輯
- `*` = 有未儲存的變更
- `✓` = 已發布
- `○` = 草稿（已儲存）

---

### 2. 搜尋功能

**快捷鍵：** `Ctrl+F`

**搜尋範圍：**
- 文章標題
- 系列名稱
- 標籤

**智能展開：**
搜尋時會自動展開包含匹配結果的系列分組。

**清除搜尋：** 按 `Escape`

---

### 3. 視圖選項

點擊右上角的 ⚙️ 圖示：

```
☑ 依系列分組
☑ 顯示狀態圖示
───────────────
全部展開
全部收合
```

**設定持久化：**
您的選擇會自動儲存到本地，下次開啟時保持相同設定。

---

### 4. 摺疊/展開

**點擊系列標題**即可摺疊或展開該系列。

**快速操作：**
- 全部展開：查看所有文章
- 全部收合：只顯示系列標題，節省空間

---

## 🎯 使用場景

### 場景 1：編輯系列文章

```
1. 展開「Vue 3 進階教學」系列
2. 看到當前編輯的是 #3
3. 想參考 #2 的內容
4. 點擊 #2 快速切換
```

### 場景 2：找到特定文章

```
1. 按 Ctrl+F 開啟搜尋
2. 輸入 "hook"
3. 自動展開 React 系列
4. 看到「Hook 實戰」文章
5. 點擊切換
```

### 場景 3：查看整體結構

```
1. 點擊「全部展開」
2. 看到所有文章的組織結構
3. 系列文章按順序排列
4. 獨立文章按修改時間排序
```

## 📊 效能優化

**虛擬滾動：**
當文章超過 100 篇時，會啟用虛擬滾動，確保流暢體驗。

**延遲載入：**
摺疊的系列不會渲染子項目，減少 DOM 節點。

**記憶化：**
分組結果會被快取，避免重複計算。

## 💡 使用技巧

### 技巧 1：組織系列文章

在 Frontmatter 中設定：
```yaml
series: Vue 3 進階教學
seriesOrder: 3
```

文章會自動歸類到「Vue 3 進階教學」系列的第 3 位。

### 技巧 2：快速切換

按住 `Ctrl` 點擊文章，會在新分頁開啟（未來功能）。

### 技巧 3：批量操作

選中系列標題，右鍵可以批量操作該系列的所有文章（未來功能）。

## 🔄 切換回舊版

如果您更喜歡卡片式列表：

1. 在 `src/App.vue` 中
2. 將 `<ArticleListTree />` 改回 `<ArticleList />`
3. 重新啟動應用
