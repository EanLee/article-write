# Architecture Decision Records (ADR)

本目錄記錄專案中重要的架構決策。

## 格式

每個 ADR 檔案命名為 `ADR-NNNN-簡短描述.md`，包含：

- **狀態**：Proposed / Accepted / Deprecated / Superseded
- **背景**：決策的情境與問題
- **決策**：做了什麼決定
- **後果**：決策帶來的影響
